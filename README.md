 👥 KeenKeeper

**KeenKeeper** is your personal vault of meaningful connections. It is a relationship management application designed to help you ensure, track, and nurture the relationships that matter most. Stay on top of your friendships so nobody slips through the cracks.

 🚀 Technologies Used
- **Next.js 16 (App Router)** - For super-fast routing and modern React architecture.
- **Tailwind CSS v4** - For rapid and pixel-perfect UI styling matching the Figma references.
- **Recharts** - For rendering beautiful analytics and interaction PIE charts.
- **React Hot Toast** - For elegant, accessible user interaction notifications.
- **Lucide React** - High-quality vector icons combined with custom graphics.

 ✨ Key Features
1. **Interactive Timeline Logs**: Automatically records whenever you Quick Check-in with a friend via Call, Text, or Video, establishing a permanent history of your interactions.
2. **Intelligent Status Tiers**: Visually surfaces friends as "on-track", "almost due", or "overdue" based on custom contact goals assigned individually to each person!
3. **Friendship Analytics**: Delivers an at-a-glance pie charts showing standard interaction metrics directly on the Stats dashboard.
